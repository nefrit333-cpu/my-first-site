# Как участвовать в проекте

Этот проект создан как учебный сайт-портфолио frontend-разработчика.

Сейчас проект развивается в учебном формате, но структура репозитория постепенно приближается к реальному frontend-проекту.

## Цель проекта

Основная цель проекта — практиковаться в:

- HTML
- CSS
- JavaScript
- Git
- GitHub
- GitHub Pages
- работе с документацией
- настройке качества кода
- оформлении frontend-проектов

## Как запустить проект локально

Проект является статическим сайтом.

Для просмотра сайта достаточно открыть файл:

```text
index.html
```
