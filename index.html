Обо мне

Я развиваюсь как frontend-разработчик

Я изучаю веб-разработку с нуля и шаг за шагом собираю своё первое портфолио. Сейчас мой основной фокус — уверенно освоить HTML, CSS, JavaScript, адаптивную верстку, работу с Git, GitHub и публикацию сайтов через GitHub Pages.

Для меня важно не просто копировать код, а понимать, как устроена страница: где находится структура, где стили, как работает интерактивность и как изменения попадают из локального проекта в живую версию сайта.

Моя ближайшая цель — собрать несколько аккуратных учебных проектов, научиться уверенно дорабатывать интерфейсы и подготовиться к переходу на React и Next.js.

Навыки

Что я изучаю

HTML

HTML отвечает за структуру страницы: заголовки, текст, списки, кнопки, ссылки, изображения и формы.

CSS

CSS отвечает за внешний вид сайта: цвета, размеры, отступы, карточки, тени, шрифты и адаптивность.

JavaScript

JavaScript отвечает за поведение сайта: клики, формы, открытие меню, интерактивные элементы и работу с данными.
