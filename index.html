Я развиваюсь как frontend-разработчик и собираю портфолио через практику: улучшаю структуру страниц, работаю со стилями, добавляю интерактивность и публикую результат онлайн через GitHub Pages.

FAQ

Частые вопросы

Да, простой статичный сайт можно сделать на HTML и CSS. Но для интерактива: меню, формы, модальные окна, фильтры и отправка данных — нужен JavaScript.

JavaScript отвечает за поведение сайта. Он позволяет реагировать на клики, проверять формы, открывать окна, менять контент и работать с данными.

К React лучше переходить после уверенной базы: HTML, CSS, адаптив, JavaScript, DOM, события, формы и работа с массивами.

Да, но его нужно доработать: добавить реальные проекты, ссылки на GitHub, описание задач, скриншоты, адаптив и аккуратный код.
