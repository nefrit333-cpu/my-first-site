## Основные разделы сайта

### Главный экран

Первый экран представляет меня как frontend-разработчика в обучении и показывает текущий фокус: HTML, CSS, JavaScript, Git и GitHub Pages.

### Обо мне

Раздел рассказывает о моём пути в веб-разработке, целях обучения и текущем фокусе.

### Навыки

В блоке навыков показано, какие технологии я изучаю и как применяю их в проекте.

### Проекты

В разделе проектов собраны первые учебные проекты. Основной проект — этот сайт-портфолио.

### FAQ

Раздел с частыми вопросами реализован в формате раскрывающегося аккордеона.

### Контакты

Форма контактов пока работает как учебная: проверяет поля и показывает сообщение об успешной отправке.

### Релиз v1.0

Финальный блок показывает, что первая версия сайта уже опубликована и оформлена как учебный релиз.
